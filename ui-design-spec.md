# UI设计规范 — 清新教育风

## 1. 设计原则

- **效率优先**：减少操作步骤，信息一目了然，老师每周处理120个学生不疲惫
- **降低认知负担**：高频操作（状态调整、话术编辑）触手可及，无多余装饰干扰
- **亲和专业**：像一本打开的备课笔记，有温度但不失严谨
- **清晰反馈**：每个操作都有明确的状态变化，老师知道系统正在做什么

## 8. 组件规范

### 8.1 按钮

**主按钮（Primary）**
- 背景：`#4A90D9` → hover: `#357ABD` → active: `#2B6BA8`
- 文字：`#FFFFFF`，14px，medium
- padding：`8px 20px`
- 圆角：6px
- 阴影：浅阴影

**次要按钮（Default）**
- 背景：`#FFFFFF`
- 边框：`1px solid #D9D9D9` → hover: `#4A90D9`
- 文字：`#595959` → hover: `#4A90D9`，14px
- padding：`8px 20px`
- 圆角：6px

**文字按钮（Text）**
- 背景：透明
- 文字：`#4A90D9`，14px
- hover：文字加下划线，背景出现 `#E8F4FD`

**危险按钮（Danger）**
- 背景：`#F5222D` → hover: `#CF1322`
- 文字：`#FFFFFF`

### 8.2 输入框

- 背景：`#FFFFFF`
- 边框：`1px solid #D9D9D9`
- 圆角：4px
- padding：`8px 12px`
- placeholder：`#BFBFBF`，14px
- 聚焦：边框变为 `#4A90D9`，外发光 `0 0 0 3px rgba(74,144,217,0.15)`
- 禁用：背景 `#F5F5F5`，文字 `#BFBFBF`

### 8.3 表格

- 表头：背景 `#FAFAFA`，文字 `#262626`，14px，medium，上下padding 12px
- 表体行：背景 `#FFFFFF`，上下padding 14px
- 行hover：背景 `#F5F7FA`
- 行选中：背景 `#E8F4FD`
- 分割线：仅底部 `1px solid #F0F0F0`，无竖线
- 操作列：文字按钮（编辑、复制、删除），hover显示
- 空状态：居中插画+"暂无数据"文字

### 8.4 状态标签（Tag）

| 状态 | 背景色 | 文字色 | 示例 |
|------|--------|--------|------|
| 已生成 | `#E8F4FD` | `#4A90D9` | 已生成 |
| 已编辑 | `#F6FFED` | `#52C41A` | 已编辑 |
| 待调整 | `#FFFBE6` | `#FAAD14` | 待调整 |
| 已导出 | `#F5F5F5` | `#8C8C8C` | 已导出 |
| 缺勤 | `#FFF1F0` | `#F5222D` | 缺勤 |
| 优秀 | `#F6FFED` | `#52C41A` | 优秀 |

- 圆角：999px（全圆角）
- padding：`2px 10px`
- 字号：12px，medium

### 8.5 卡片

- 背景：`#FFFFFF`
- 圆角：8px
- 阴影：`0 1px 2px rgba(0,0,0,0.05)`
- padding：24px
- 标题区：底部边框 `1px solid #F0F0F0`，padding-bottom 16px，margin-bottom 16px

### 8.6 弹窗（Modal）

- 背景遮罩：`rgba(0,0,0,0.45)`
- 弹窗背景：`#FFFFFF`
- 圆角：8px
- 阴影：深阴影
- 宽度：根据内容自适应，最小400px，最大720px
- 头部：标题16px bold，右侧关闭按钮（×）
- 底部操作区：居右，主按钮+次要按钮，间距12px
- 内容区：padding 24px，max-height 60vh，超出滚动

### 8.7 提示消息（Message / Toast）

- 成功：左侧竖条 `#52C41A`，背景 `#FFFFFF`，阴影中阴影
- 警告：左侧竖条 `#FAAD14`
- 错误：左侧竖条 `#F5222D`
- 位置：页面顶部居中，距顶部24px
- 自动消失：3秒，带滑出动画

## 10. 交互状态与动画

### 10.1 状态切换过渡
- 标签状态变化：背景色渐变 200ms
- 表格行hover：背景色渐变 150ms
- 按钮hover：颜色渐变 150ms，阴影加深

### 10.2 操作反馈
- 点击按钮：按钮scale(0.98) 50ms，释放恢复
- 复制成功：Toast 从顶部滑入，3秒后滑出
- 保存成功：该行短暂闪烁 `#E8F4FD` → `#FFFFFF`，提示"已保存"
- 加载中：按钮内文字替换为旋转图标，禁用点击

### 10.3 页面切换
- 路由切换：内容区淡入淡出 200ms
- 弹窗打开：遮罩淡入 + 弹窗从上方滑入 300ms
- 弹窗关闭：弹窗向下滑出 + 遮罩淡出 200ms

## 13. 前端技术实现建议

- **UI框架**：Element Plus（与Vue3配套，主题定制成熟）
- **主题定制**：通过 Element Plus 的 CSS Variables 覆盖默认变量
- **布局**：Flexbox + Grid，响应式使用 CSS Media Queries
- **动画**：CSS Transitions 为主，复杂动画使用 Vue `<Transition>`
- **图标**：`@phosphor-icons/vue` 或 `@heroicons/vue`
- **字体**：系统默认中文字体栈，无需额外引入
