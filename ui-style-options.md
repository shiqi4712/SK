# UI界面设计风格选项

## 选型原则

- **效率优先**：老师每周处理120个学生，界面必须减少操作步骤，信息一目了然
- **降低认知负担**：话术编辑、状态调整是高频操作，界面不能花哨干扰
- **亲和力**：面向少儿教培，过于冰冷的工业风不合适
- **适配场景**：老师可能在平板、笔记本、台式机上使用，需响应式兼容

---

## 方案一：清新教育风（推荐）

**整体感受**：干净、明亮、有温度，像打开一本备课笔记

**配色**
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #4A90D9 | 清澈蓝，沉稳专业 |
| 辅助色 | #52C41A | 生机绿，用于成功/完成状态 |
| 警示色 | #FAAD14 | 温和橙，用于提醒/待办 |
| 错误色 | #F5222D | 标准红，错误/缺勤 |
| 背景色 | #F5F7FA | 浅灰蓝，护眼不刺眼 |
| 卡片背景 | #FFFFFF | 纯白，内容区 |
| 文字主色 | #262626 | 深灰，非纯黑更柔和 |
| 文字次要 | #8C8C8C | 中灰，辅助信息 |

**布局特点**
- 左侧导航栏 + 右侧内容区，经典后台布局
- 导航栏用主色渐变或纯白+主色点缀
- 内容区大量留白，卡片式信息分组
- 表格行高充足，鼠标悬停高亮当前行

**组件风格**
- 按钮：圆角6px，主按钮填充色，次要按钮描边
- 标签：圆角 pill 形状，状态标签带浅色背景
- 输入框：聚焦时主色边框发光
- 表格：斑马纹或纯白行+分割线，操作按钮 hover 显示

**适用场景**：追求专业感与亲和力平衡的教培机构，老师群体偏年轻

---

## 方案二：极简效率风

**整体感受**：像 Notion / Linear，信息密度高，操作路径极短

**配色**
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #2563EB | 经典蓝，专注感 |
| 辅助色 | #10B981 | 翠绿，行动确认 |
| 警示色 | #F59E0B | 琥珀色 |
| 错误色 | #EF4444 | 柔和红 |
| 背景色 | #FAFAFA | 极浅灰 |
| 卡片背景 | #FFFFFF | 纯白 |
| 文字主色 | #171717 | 近黑 |
| 文字次要 | #737373 | 中灰 |

**布局特点**
- 顶部导航栏（极简图标+文字）+ 全宽内容区
- 隐藏侧边栏或收起为图标栏，最大化内容空间
- 列表页信息密度高，减少翻页
- 编辑操作 inline（行内直接编辑，无需跳转）

**组件风格**
- 按钮：圆角4px，扁平设计，hover 微上移+阴影
- 标签：小圆角矩形，无底色纯文字或极浅底色
- 输入框：无边框底线条风格，或极简方框
- 表格：无竖线，仅横线分隔，极简风格

**适用场景**：老师操作熟练后追求极致效率，或需要在较小屏幕上操作

---

## 方案三：温暖亲和风

**整体感受**：像钉钉/企业微信，亲切友好，降低老师使用焦虑

**配色**
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #FF8C42 | 暖橙，活力亲和 |
| 辅助色 | #6BCB77 | 柔和绿 |
| 警示色 | #FFD93D | 明亮黄 |
| 错误色 | #FF6B6B | 珊瑚红 |
| 背景色 | #FFF9F5 | 极浅暖色 |
| 卡片背景 | #FFFFFF | 纯白 |
| 文字主色 | #2D3436 | 深炭灰 |
| 文字次要 | #A0A0A0 | 浅灰 |

**布局特点**
- 左侧导航栏圆角卡片化，像手机App的抽屉导航
- 内容区卡片带轻微圆角和柔和阴影
- 图标使用线性图标+主色点缀，整体偏圆润
- 空状态、提示使用插画而非纯文字

**组件风格**
- 按钮：圆角8px甚至更大，饱满亲和
- 标签：大圆角，像小药丸
- 输入框：圆角较大，带内部图标提示
- 表格：行圆角，行间有微小间距（非贴在一起）

**适用场景**：老师年龄偏大或对技术产品不熟悉，需要更友好的视觉引导

---

## 方案四：深色专注风

**整体感受**：像 VS Code / Figma 深色模式，夜晚使用不刺眼，内容突出

**配色**
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #60A5FA | 亮蓝，深色背景上更醒目 |
| 辅助色 | #34D399 | 翠绿 |
| 警示色 | #FBBF24 | 金黄 |
| 错误色 | #F87171 | 浅红 |
| 背景色 | #0F172A | 深蓝黑 |
| 卡片背景 | #1E293B | 深灰蓝 |
| 文字主色 | #F1F5F9 | 近白 |
| 文字次要 | #94A3B8 | 灰蓝 |

**布局特点**
- 深色背景下内容对比度高，视觉焦点集中
- 适合长时间编辑话术，减少眼睛疲劳
- 高亮状态使用发光效果
- 侧边栏与内容区层次通过深浅色区分

**组件风格**
- 按钮：深色背景上带发光边框或渐变
- 标签：深色底色+亮色文字
- 输入框：深色背景，聚焦时主色光晕
- 表格：深灰行交替，hover 行提亮

**适用场景**：老师常在晚上备课回访，或偏好深色模式的用户

---

## 推荐选择

| 你的情况 | 推荐方案 |
|---------|---------|
| 追求稳妥、大多数老师能接受 | **方案一：清新教育风** |
| 老师年轻、熟悉互联网产品 | **方案二：极简效率风** |
| 老师群体偏大、需要降低学习成本 | **方案三：温暖亲和风** |
| 老师常晚上使用、或系统想差异化 | **方案四：深色专注风** |
| 不确定，想兼容所有人 | **方案一 + 提供深色模式切换** |

**最终建议**：采用**方案一（清新教育风）为主体**，在设置中提供**浅色/深色模式切换**。这样默认状态下专业亲和，有需要的老师可切到深色模式护眼。

## 技术实现建议

- 使用 Element Plus / Ant Design Vue 等组件库，通过主题定制实现配色
- CSS 变量定义颜色体系，方便切换浅色/深色模式
- 响应式断点：>=1440px(大屏) / >=1280px(笔记本) / >=768px(平板) / <768px(手机，仅适配登录页)
- 图标库：推荐使用 Phosphor Icons 或 Heroicons，线条干净
